Installation Guide

Follow these steps to install BackupCraft and keep your Minecraft worlds safe.

How to Install BackupCraft

  1. Download the latest version from the GitHub repository.
  2. Extract the downloaded archive (.zip or .tar.gz) anywhere you like.
  3. Open a terminal and navigate to the extracted folder:
    4. cd ~/Downloads/BackupCraft
  4. Run the installation script:
    5. ./install.sh
  5. The installer will automatically:
    • Install all required dependencies (like 7zip, jq, etc.);
    • Move the necessary files to your ~/ home directory;
    • Make BackupCraft ready to use right away.
  6. Once installed, simply run the program from the terminal using: 
    7. ./bchub.sh
